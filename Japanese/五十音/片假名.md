# 片假名

## ta 行

| ta行假名     | 浊化        |                                                        |
| ------------ | ----------- | ------------------------------------------------------ |
| **タ** (ta)  | **ダ** (da) |                                                        |
| **チ** (chi) | **ヂ** (ji) | 可以看到原本的 di 被 ji占用了，使用 "**ディ**" 表示 di |
| **ツ** (tsu) | **ヅ** (zu) |                                                        |
| **テ** (te)  | **デ** (de) |                                                        |
| **ト** (to)  | **ド** (do) |                                                        |

ti 被 chi 占用了，日语用 "**ティ**" 表示 ti

eg：

- ディズニー（Disney）
- ディナー（Dinner）
- パーティー (Party)
